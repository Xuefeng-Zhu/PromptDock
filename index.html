FAQs

Small app, useful defaults.

Does PromptDock require an account?

No. You can try PromptDock without an account, then sign in when you want your prompt recipes synced across devices.

Can I use it on Windows and Mac?

Yes. PromptDock can be used on Windows and Mac, and the hosted web app is available from any modern browser.

Can it run outside Tauri?

Yes. The browser runtime falls back to localStorage and web clipboard APIs. You can open the hosted app at promptdock-95e31.web.app.

Where do I download it?

Builds and packaged artifacts live on the GitHub Releases page.
